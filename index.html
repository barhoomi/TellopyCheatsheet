DJITelloPy Cheat Sheet

A quick reference for the Ryze Tello drone library.

Connection & Setup

tello = Tello()Creates a Tello instance.
tello.connect()Enters SDK mode. Must be called first.
tello.end()Closes connection. Call at the end.
tello.reboot()Reboots the drone.

Flight & Emergency

tello.takeoff()Auto takeoff.
tello.land()Auto landing.
tello.emergency()Stops all motors immediately.
tello.initiate_throw_takeoff()Enables throw-to-takeoff.

Flip Commands

tello.flip_left()Flip to the left.
tello.flip_right()Flip to the right.
tello.flip_forward()Flip forward.
tello.flip_back()Flip backward.

Basic Usage Example

from djitellopy import Tello
tello = Tello()

tello.connect()
print(tello.get_battery())

tello.takeoff()
tello.move_forward(50)
tello.land()
tello.end()
